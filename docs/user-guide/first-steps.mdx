## Welcome to Llumen!

You've installed llumen—let's get you chatting.

## Initial Login

1. Open your browser to `http://localhost` (or your configured address)
2. You'll see the login screen
3. Enter default credentials:
   - **Username:** `admin`
   - **Password:** `P@88w0rd`

<Warning>
  **Important:** Change the default password immediately!  
  Go to Settings → Security → Change Password
</Warning>

### Main Components

**1. Chat Area**
- Center of the screen
- Shows conversation history
- Messages render in real-time

**2. Input Box**
- Bottom of screen
- Type your messages here
- Access action menu for modes and attachments

**3. Settings**
- Top-right corner (gear icon)
- Configure appearance, API, and behavior

**4. Sidebar**
- Conversation history
- Create new chats
- Access past conversations

## Your First Message

1. Click in the input box
2. Type a simple question:
   ```
   Hello! What can you help me with?
   ```
3. Press Enter or click Send
4. Watch the response stream in real-time

<Note>
  First message might take a moment as the API connection initializes.
</Note>

## Exploring Chat Modes

### Try Normal Chat

Default mode. Great for general questions.

```
Explain how HTTP works in simple terms
```

### Try Web Search

Click the action menu (icon in input box) and select "Web Search":

```
What are the latest features in Rust?
```

Notice how it provides sources!

### Try Deep Research

Select "Deep Research" mode:

```
Compare different self-hosted AI chat solutions
```

Watch as it creates a research plan and investigates thoroughly.

## Working with Media

### Upload a PDF

1. Click the attachment icon
2. Select a PDF file
3. Wait for processing
4. Ask questions:
   ```
   Summarize the main points of this document
   ```

### Using LaTeX

Type mathematical expressions:

```
Explain the equation: $E = mc^2$

Show me the quadratic formula in LaTeX
```

## Customizing Your Experience

### Change Theme

1. Open Settings (gear icon)
2. Go to Appearance
3. Choose from:
   - Llumen (default)
   - Ocean (blue)
   - Sunset (warm)
   - Variations with patterns

### Adjust Settings

**Chat Behavior:**
- Enable/disable streaming
- Auto-scroll
- Message formatting

**API Configuration:**
- Select preferred models
- Adjust temperature
- Set max tokens

## Conversation Management

### Starting Fresh

- Click "New Chat" in sidebar
- Previous conversation is automatically saved

### Accessing History

- Click any conversation in sidebar
- All messages are preserved
- Resume where you left off

### Organizing Conversations

- Conversations auto-title based on content
- Delete unwanted conversations
- Export important chats

## Keyboard Shortcuts

| Shortcut | Action |

| `Enter` | Send message |

| `Ctrl/Cmd + K` | New conversation |

## Tips for Better Results

### Be Specific

❌ **Vague:**
```
Tell me about programming
```

✅ **Specific:**
```
Explain the difference between async/await and promises in JavaScript
```

### Provide Context

❌ **No context:**
```
How do I fix this?
```

✅ **With context:**
```
I'm getting a "connection refused" error when trying to connect to my Docker container from the host. How do I fix this?
```

### Use the Right Mode

- **Normal:** Quick questions, general chat
- **Web Search:** Current events, facts
- **Deep Research:** Complex topics, thorough analysis

### Iterate

Don't be afraid to:
- Ask follow-up questions
- Request clarification
- Ask for examples
- Request different formats
