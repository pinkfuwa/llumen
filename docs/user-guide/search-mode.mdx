## Overview

Web Search mode combines AI intelligence with real-time web data.

## Activating Search Mode

1. Click the action menu icon in chat input
2. Select "Web Search"
3. Mode indicator appears in input area

## How It Works

1. You ask a question
2. Llumen searches the web for relevant information
3. Results are gathered and analyzed
4. AI synthesizes an answer with citations
5. Sources are displayed with the response

## Best Use Cases

### Current Events

```
What's the latest news about AI regulation in the EU?
```

### Product Information

```
Compare the specs of the latest MacBook Pro models
```

### Fact Checking

```
What's the current population of Tokyo?
```

### Trending Topics

```
What are developers saying about Rust vs Go in 2024?
```

## Tips for Better Results

### Be Specific with Dates

```
What were the major tech announcements in January 2024?
```

### Ask for Comparisons

```
Compare the features of Docker vs Podman
```

### Request Sources

```
What are the official installation methods for Python 3.12?
Please cite official sources.
```

## Understanding Citations

- Links appear beneath the response
- Click to view original sources
- Verify important information at the source

## Limitations

- Search quality depends on available results
- Very recent events may not be indexed
- Regional content may vary
- Always verify critical information
