## Overview

Llumen features gorgeous, mobile-first themes designed for comfortable extended use. Each theme has been carefully crafted for readability and aesthetics.

## Available Themes

The application includes three hand-crafted color palettes, each available in light and dark modes:

- **llumen** — Warm orange-accented default theme
- **dracula** — Purple-accented theme inspired by the Dracula color scheme
- **vitesse** — Green-accented theme inspired by Vitesse for Vite/VSCode

All themes support a toggleable chat background pattern.

## Theme Features

### Mobile-First Design

All themes are optimized for mobile devices:
- **Touch-friendly**: Large tap targets, swipe gestures
- **Responsive**: Adapts perfectly to any screen size
- **Gesture**: Optimized gesture
- **Readable**: Carefully chosen typography and spacing

### Accessibility

All themes support:
- Keyboard navigation
- Screen readers
- Reduced motion preferences
- Custom font sizing
