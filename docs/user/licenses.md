## Third-Party Licenses

llumen uses many open-source libraries and dependencies. We are grateful to all the maintainers and contributors of these projects.

### Where to Find License Information

Complete license information for all third-party dependencies is available in the repository:

- **Frontend Dependencies**: [frontend/THIRDPARTY.txt](../../frontend/THIRDPARTY.txt)
- **Backend Dependencies**: [backend/THIRDPARTY.toml](../../backend/THIRDPARTY.toml)

These files include:
- Full list of all dependencies
- License type for each dependency
- Complete license texts
- Author and repository information

### License Types Used

All third-party dependencies use permissive open-source licenses, including:

- **MIT License** - Permissive license with minimal restrictions
- **Apache License 2.0** - Permissive license with patent protection
- **BSD Licenses** - Family of permissive licenses
- **ISC License** - Simplified permissive license similar to MIT

## In Binary Distributions

If you downloaded a pre-built binary or Docker image of llumen, license information is included:

- A `LICENSE` file in the frontend build references the full license files
- The files are accessible at:
  - `https://github.com/eason/llumen/blob/main/frontend/THIRDPARTY.txt`
  - `https://github.com/eason/llumen/blob/main/backend/THIRDPARTY.toml`
