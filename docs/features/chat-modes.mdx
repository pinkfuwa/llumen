## Overview

Llumen provides three distinct chat modes, each optimized for different use cases. Switch between them seamlessly based on your needs.

## Normal Chat Mode

**Best for:** General conversations, quick questions, creative writing, coding help

Normal mode is your standard AI assistant. It's fast, direct, and perfect for most interactions.

### Features
- Direct LLM responses
- Fastest response times 
- Context-aware conversations
- Multi-turn dialogue support

### Use Cases
- Ask questions about any topic
- Get help with coding
- Write and edit content
- Brainstorm ideas
- Explain complex concepts

### Example Prompts
```
Help me write a function to parse JSON in Rust

Explain quantum computing in simple terms

Review this email for tone and clarity
```

## Web Search Mode

**Best for:** Current events, fact-checking, research with sources

Web Search mode augments the AI with real-time web data. Get answers backed by current information and see your sources.

### Features
- Real-time web search
- Source citations
- Up-to-date information
- Fact verification

### Use Cases
- Check current news and events
- Find recent product information
- Verify facts and figures
- Research trending topics
- Compare prices and reviews

### How It Works
1. You ask a question
2. Llumen searches the web
3. Relevant information is gathered
4. AI synthesizes an answer with sources
5. You get cited, current information

### Example Prompts
```
What are the latest features in Rust 1.75?

Compare the specs of iPhone 15 Pro vs Samsung S24

What's the current situation with [topic]?
```

## Deep Research Mode

**Best for:** Complex questions, thorough analysis, multi-step research

Deep Research mode deploys autonomous agents to conduct comprehensive research. The AI plans its approach, gathers information systematically, and provides detailed findings.

### Features
- Autonomous research agents
- Multi-step investigation
- Comprehensive analysis
- Structured findings
- Progress transparency

### Use Cases
- Market research
- Technical deep-dives
- Academic research
- Competitive analysis
- Strategic planning

### How It Works
1. You pose a complex question
2. Agent creates a research plan
3. Systematically gathers information
4. Analyzes and synthesizes findings
5. Presents comprehensive report

### Example Prompts
```
Analyze the state of Rust async runtimes in 2024

Research best practices for building documentation sites

What are the tradeoffs between different self-hosted LLM options?
```

<Note>
  Deep Research mode takes longer but provides much more thorough analysis. Use it when you need depth over speed.
</Note>

## Switching Between Modes

Access the mode selector through the action menu in the chat input area. Your conversation history is preserved when switching modes.
